# Asistente LOKITS Web

Sitio web oficial de Asistente LOKITS, un asistente de WhatsApp para clanes.

## Tecnología inicial

- HTML
- CSS
- JavaScript
- Live Server para pruebas locales

## Estado

Proyecto iniciado por fases.

## Nota importante

La web está separada completamente del bot de WhatsApp.  
El ZIP del bot solo se usa como referencia para identificar comandos y funcionalidades públicas.

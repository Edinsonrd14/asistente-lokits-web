Beneficios

Todo pensado para administrar mejor el clan.

El objetivo es que los líderes y miembros puedan consultar información importante sin salir de WhatsApp y sin revisar todo manualmente.

⚔️

Control de guerra

Consulta el estado de guerra, ataques usados, pendientes, estrellas y destrucción del clan.

🏆

Seguimiento CWL

Revisa rankings, resumen de liga, desempeño por jugador y datos importantes de la temporada.

🔔

Avisos automáticos

Recibe recordatorios y alertas para ayudar a que el clan no pierda ataques importantes.

📊

Reportes claros

Obtén reportes mensuales, asistencia, actividad y datos útiles para tomar mejores decisiones.

Cómo funciona

Activación sencilla para empezar rápido.

La web solo presenta el servicio. El bot se mantiene separado y se configura de forma independiente para cada grupo de WhatsApp.

Solicitas información del servicio y del plan que necesitas.

02

Se revisa tu clan

Se confirma el clan, grupo y las funciones que quieres activar.

03

Se configura el asistente

El bot queda listo para responder comandos dentro del grupo.

FAQ

Dudas más comunes

Las preguntas están organizadas para que el cliente entienda rápido cómo funciona el servicio y qué puede esperar antes de activarlo.

¿Qué es Asistente LOKITS?

Asistente LOKITS es un asistente de WhatsApp para clanes. Permite consultar información de guerra, CWL, reportes, avisos automáticos, capital del clan, jugadores y otros datos usando comandos dentro del grupo.

¿El bot juega por los miembros?

No. El asistente no juega, no ataca y no toma decisiones dentro del juego. Su función es mostrar información, reportes, avisos y datos útiles para que el clan se organice mejor.

¿Se instala dentro de WhatsApp?

El asistente funciona dentro de grupos de WhatsApp mediante comandos usando únicamente el ID del clan, sin necesidad de introducir datos personales ni instalar aplicaciones adicionales.

¿Qué comandos incluye?

Incluye comandos como !war, !attacks, !cwltop, !cwlsummary, !rival, !capital, !player, !month, !attendance, !status, !help y otros comandos de configuración y avisos automáticos.

¿Qué son los avisos automáticos?

Son mensajes que el bot puede enviar sin que alguien escriba un comando. Por ejemplo: cuando se detecta una guerra, cuando está por iniciar, cuando inicia, cuando queda poco tiempo, cuando hay ataques aliados, ataques del rival o cambios de miembros.

¿Cuánto cuesta el servicio?

El plan base es de $3 al mes para un clan en un grupo de WhatsApp. Los grupos adicionales del mismo usuario tienen un costo de $1 al mes. También se puede consultar un plan personalizado según el caso.

¿Se publican datos privados del asistente?

No. No se publican archivos internos, sesiones de WhatsApp, credenciales, tokens, archivos .env, configuraciones privadas ni datos sensibles del servidor.

¿Asistente LOKITS está afiliado con Supercell?

No. Asistente LOKITS es una herramienta independiente para la organización de clanes y no está afiliada oficialmente con Supercell. La mayoría de la información utilizada proviene de la API oficial de Supercell.

¿Existe riesgo de baneo por usar el bot?

No. El uso de Asistente LOKITS no implica riesgo de baneo, ya que no modifica el juego, no automatiza cuentas dentro del juego ni juega en nombre de los usuarios. Solo funciona como herramienta externa de organización utilizando información pública de la API oficial de Supercell.

¿Cómo se activa el servicio?

Primero se contacta por WhatsApp, se confirma el clan, el grupo y el plan. Luego se realiza la configuración inicial y se prueban los comandos principales para verificar que todo funcione correctamente.

Qué incluye

El servicio está pensado para funcionar de forma práctica.

Además del acceso al asistente, se toma en cuenta la configuración inicial, revisión básica y soporte para que el clan pueda empezar a usarlo.

⚙️

Configuración inicial

Se vincula el clan con el grupo correspondiente y se revisa que los comandos principales funcionen correctamente.

💬

Uso desde WhatsApp

Los miembros pueden consultar información desde el grupo usando comandos simples y fáciles de recordar.

🛠️

Soporte del servicio

Se ofrece ayuda para revisar dudas, ajustes básicos y detalles de funcionamiento del asistente.

🔒

Separado del asistente

La página web es independiente. No se publican sesiones, credenciales, archivos internos ni configuraciones privadas.
